## Research Note — NLP on Federal Reserve Communications

<div align="center">
<sub>FIXED INCOME · MONETARY POLICY SIGNALING · NATURAL LANGUAGE PROCESSING</sub>
</div>

<br/>

**Objective.** Quantify the hawkish/dovish stance of FOMC press conferences to anticipate monetary policy inflections ahead of rate decisions.

**Analytical Framework.**

```
Transcripts (FOMC)
       │
       ▼
Preprocessing → Tokenization · TF-IDF · Domain Stopword Removal
       │
       ▼
Embedding Layer (dim=100) → Fine-tuned BERT [FinancialPhraseBank calibration]
       │
       ▼
BiLSTM / Deep CNN (TensorFlow) → Per-session Sentiment Score
       │
       ▼
Statistical Validation → Wilcoxon Signed-Rank · Bonferroni Correction
```

**Key Finding.** Dovish lexical clusters identified via Bag-of-Words exhibit a statistically significant lead of 2–3 sessions ahead of rate cuts (*p* < 0.01), providing an actionable signal for fixed income positioning.

→ [Access repository](https://github.com/maxime2476/sentiment-powell-nlp)

## Repository Architecture Standard

All repositories follow a reproducible research structure aligned with institutional data science norms:

```
project/
├── README.md           ← Executive summary · Results · Setup guide
├── notebooks/          ← Numbered EDA & modelling iterations (01_eda.ipynb …)
├── src/                ← Modular production code · Utility pipelines
├── data/               ← Anonymised samples only — raw data excluded via .gitignore
├── report/             ← Final PDFs · High-resolution figures
├── dashboard/          ← Streamlit / Dash application · Dockerfile
├── requirements.txt    ← Pinned dependency manifest
└── LICENSE
```

> Data security is non-negotiable. No proprietary datasets, client records, or API credentials are committed to public repositories.
